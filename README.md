# Rock Festival 

## Description

This project is a responsive landing page for a  Rock festival.
The purpose of the website is to promote the festival, present upcoming dates, highlight themed zones and gallery content, and create a visually attractive interface for visitors.

The page focuses on semantic HTML structure and modern CSS layout techniques, ensuring readability, responsiveness, and a clean user experience.

---

## Technologies Used

* HTML5 (Semantic Structure)
* CSS3 (Flexbox, Grid Layout, Media Queries)
* No JavaScript (HTML/CSS only)

## Features

* Semantic HTML structure (header, main, section, footer)
* Hero section with strong visual impact and CTA
* Events section with structured HTML table (thead, tbody)
* About section with descriptive content
* Unordered list (ul) for festival experiences
* Themed zones section using card layout
* Multimedia gallery using CSS Grid
* Responsive layout using media queries
* Navigation adapted for mobile view
* Clean and organized UI

---

## Responsive Design

The page adapts to different screen sizes:

* Mobile devices
* Desktop screens (>1024px)

Media queries are used to adjust:

* Layout (Grid/Flex)
* Typography
* Spacing
* Navigation

## How to Run the Project

### Option 1 – Open directly in browser (Recommended)

1. Download or clone the project repository:

```bash
git clone https://github.com/JAHN77/prueba-htmal-css
```

2. Open the project folder.

3. Locate the file:

```bash
index.html
```

4. Double-click the file OR right-click → "Open with browser".

The page will load automatically.

---

### Option 2 – Using Live Server (Recommended for development)

If you are using VS Code:

1. Install the extension: Live Server

2. Open the project folder in VS Code.

3. Right-click on `index.html`

4. Select "Open with Live Server".

The page will open in your default browser with live reloading.
